# Gaegane Group — Final Homepage Concept Samples

This package contains 3 homepage / hero section concepts built for client review.

## Concepts
- `concept-1-corporate.html` — clean corporate premium
- `concept-2-bold.html` — bold modern direction
- `concept-3-minimal.html` — minimal luxury direction

## Files
- `index.html` — concept selector page
- `shared.css` — shared styling
- `images/` — extracted and selected real images from the company profile

## Notes
These concepts use real company content and extracted visuals from the uploaded profile document so the client can choose a direction that feels closest to the brand.
