# Gaegane Group — Hero Concept Samples

This package contains 3 different homepage / hero section directions for client review.

## Files
- `index.html` — selector page
- `concept-1-corporate.html` — clean corporate premium direction
- `concept-2-bold.html` — dark modern bold direction
- `concept-3-minimal.html` — minimal luxury direction
- `shared.css` — shared styling across the samples

## How to review
Open `index.html` in a browser and choose between the 3 concept pages.

## Notes
These are concept-only homepage samples created to help choose a design direction before the full multi-page website is built.
